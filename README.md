# X インプレゾンビ 自動フィルター【日本語バズ特化版】

X（旧Twitter）の日本語バズツイートにぶら下がる、海外インプレゾンビやAI生成の不自然な日本語リプライを自動で検出し、非表示または折りたたみ処理を行うUserscriptです。

## 🌟 主な特徴

* **海外ゾンビ撃退**: 本文やプロフィールがアラビア語、キリル文字、タイ語などの場合に高精度で判定。
* **AI生成日本語の検知**: ChatGPTやGemini等が出力しがちな、過剰に丁寧な定型文パターンをブロック。
* **不自然な日本語表現**: 日本人が使わない熟語や、絵文字のないロボット特有の文体をスコアリング。
* **スパム挙動の阻止**: 同一アカウントによる複数回リプライ、リンクカードのみ、エンゲージメントゼロなどを検知。
* **リアルタイム処理**: ページをスクロールするたびに動的にスキャンし、ゾンビを即座に排除。

## 🛠 判定ロジック（スコアリング内訳）

合計スコアが **閾値（デフォルト: 3）** 以上でフィルターが作動します。

| カテゴリ | 判定内容 | スコア |
| :--- | :--- | :---: |
| **海外アカウント** | 本文が外国語（アラビア・キリル・タイ・英語等） | +3 |
| **プロフィール** | 自己紹介欄（Bio）が日本語を含まない外国語構成 | +3 |
| **AI定型文** | AI特有の典型的な言い回し（「〜の役に立てれば幸いです」等） | +2 |
| **AI語彙** | 「貴重な情報」「深い洞察」などAIが好むキーワードの複数使用 | +1〜2 |
| **不自然な日本語** | 過剰な敬語かつ絵文字なし、または日本語偽装の海外ハンドル | +1〜2 |
| **重複リプライ** | 同一スレッド内で同じアカウントが2回以上返信（遡及処理あり） | +3 |
| **その他スパム** | 絵文字のみ、ハッシュタグ羅列、エンゲージメント完全ゼロ | +1〜2 |

## 🚀 導入方法

1. ブラウザに **Tampermonkey** などのユーザースクリプト管理拡張をインストールします。
2. 新規スクリプト作成で `impression_zombie_filter_jp.user.js` の内容を貼り付けて保存してください。
3. Xを開くと右下に 🧟 アイコンが表示され、管理パネルから設定変更が可能です。

## ⚙️ 設定パネルの機能

* **モード切替**: 「🔽 折りたたみ（クリックで展開可）」と「🙈 完全非表示」を選択可能。
* **閾値調整**: 判定の厳しさを1〜10の間で変更できます。
* **統計表示**: どの理由（スコア項目）で何件ブロックしたかの内訳をリアルタイム表示。
* **すべて復元**: 誤判定が気になる場合、一時的にすべてのフィルターを解除できます。

---
免責事項：本スクリプトはクライアントサイドで動作するため、Xの仕様変更により動作しなくなる場合があります。
